# 加群を基点とした数体系と代数構造の完全統合マップ

## 1. 基礎概念の定義：スカラーの正体

スカラーとは単なる「数」ではなく、加群に対して作用する**「環 (Ring)」**の要素です。

* **加群 (Module)**: 環 $R$（スカラー）が可換群 $M$（作用対象）に作用するシステム。
* **多元環 (Algebra)**: 自身が加群（またはベクトル空間）であり、かつ自身の中で「積（乗法・合成）」が定義されており、スカラーとしても振る舞える構造。
* **写像 (Mapping)**: 集合間の要素の対応関係。特定の構造を保つ写像（線形写像など）を「住人」として集め、演算を定義することで「環」という構造に昇格する。

---

## 2. スカラーとなり得る構造体のバリエーション

「動き（写像）」をパッケージ化して積を定義したものは、別の加群のスカラーになれます。

| スカラーとなる構造体 | 数学的構造 | 写像としての役割 | 具体的な作用対象 |
| :--- | :--- | :--- | :--- |
| **正方行列** | **全行列環** | 同一空間内の変形・回転 | ベクトル / 行列 |
| **複素数 / 四元数** | **体 / 斜体** | 2次元/4次元の回転拡大 | 複素・四元ベクトル |
| **多項式** | **多項式環** | 写像の指示書のパッケージ化 | 行列の動き（$f(T)v$ など） |
| **自己準同型写像** | **自己準同型環** | 同一構造内での自己変換 | 関数空間 / 抽象空間 |
| **十六元数** | **冪結合代数** | 高次元の非結合的な作用 | 十六元数上の加群 |

## 3. スカラー（環・体・代数）の階層構造

スカラーになれる集合を、構造の厳しさ（割り算の可否等）で分類します。

| 名前 | 集合記号 | 数学的構造 | スカラーとした時の呼称 | 特徴 |
| :--- | :---: | :--- | :--- | :--- |
| **整数** | $\mathbb{Z}$ | **可換環** | **$\mathbb{Z}$-加群** | 割り算ができない。離散的。 |
| **有理数** | $\mathbb{Q}$ | **可換体** | **ベクトル空間** | **基準点**。ここから線形代数が成立。 |
| **実数 / 複素数** | $\mathbb{R} / \mathbb{C}$ | **可換体** | **ベクトル空間** | 標準的で最も扱いやすい。 |
| **正方行列** | $M_n(R)$ | **非可換環** | **左$R$-加群への作用** | 「動き（自己準同型）」をスカラー化。 |
| **四元数** | $\mathbb{H}$ | **斜体** | **左/右ベクトル空間** | 交換法則が消失。 |
| **八元数** | $\mathbb{O}$ | **交代代数** | **非結合的加群** | 結合法則が消失。 |
| **十六元数** | $\mathbb{S}$ | **冪結合代数** | **非結合性加群** | 交代法則が消失。零因子が存在。 |

## 5. 構造体のバリエーション（操作される側・繋ぐ側）

スカラー $R$ を成分とする構造。これ自体が「環（自己準同型）」を成すかどうかが、スカラー適格性の境界です。

| 構造体 | 分類 | 写像としての性質 | スカラーになれるか |
| :--- | :--- | :--- | :--- |
| **ベクトル** | **自由加群** | されるがままの要素。 | 通常はなれない |
| **正方行列** | **全行列環** | **同一空間内の動き（自己準同型）** | **なれる** |
| **長方形行列** | **加群 / 写像** | **空間と空間を繋ぐ橋渡し** | **なれない** (積が閉じない) |
| **多項式** | **多項式環** | **操作指示の合成・レシピ** | **なれる** |
| **自己準同型写像** | **環 (Ring)** | **同一領域内での変換の集積** | **なれる** |

## 7. まとめ：基準とするべき考え方

1. **「割り算」を基準にする**: 
   スカラー側で割り算ができれば「ベクトル空間」、できなければ「加群」。この自由度が基底の存在や次元の一意性を決める。
2. **「自己準同型（自閉性）」を基準にする**: 
   正方行列や多項式のように、入り口と出口が同じ規格であり、自分たち同士で掛け算（合成）をして同じ集合に戻るルール（環）を持つものだけが、他者を操る「スカラー」になれる。
3. **「写像」の昇格条件**: 
   長方形行列のような次元を跨ぐ写像は「動き」に過ぎないが、正方行列や自己準同型写像のように「同じ空間の中での動き」をパッケージ化して演算を定義すれば、それは新しい「スカラー」として君臨する。
